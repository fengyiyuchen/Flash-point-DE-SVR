# Flash-point-DE-SVR
Dataset and DE-SVR model code for predicting flash points of binary miscible mixtures

## Project Description
This repository contains the data and code for predicting flash points (FPs) of binary miscible mixtures using a hybrid differential evolution (DE) and support vector regression (SVR) framework.

A novel data-driven QSPR approach is developed, where DE is employed to globally optimize key SVR hyperparameters, improving the modeling of nonlinear structure–property relationships in mixture systems.

## File Structure
- `data/`: Binary mixture flash point dataset (332 samples from 33 binary mixtures)
- `code/`: Python scripts for DE–SVR modeling, data preprocessing, and performance evaluation
- `README.md`: This file

## Usage
The code and dataset are provided for academic reference and reproducibility purposes. Run the scripts in the `code/` folder to reproduce the results.
